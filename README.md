# SkillCraft Technology - Web Development Internship

Welcome to my Web Development Internship project repository completed as part of the **SkillCraft Technology** Internship Program.

This repository contains all the tasks assigned during the internship, showcasing my skills in HTML, CSS, and JavaScript through interactive and responsive web applications.

---

## 🚀 Internship Details

- **Organization:** SkillCraft Technology
- **Domain:** Web Development
- **Duration:** 1 Month
- **Technology Stack:** HTML5, CSS3, JavaScript

---

## 📂 Projects Included

### 🔹 Task 1 – Responsive Landing Page
A modern and fully responsive landing page featuring:
- Fixed navigation bar
- Interactive hover effects
- Navigation color change on scroll
- Smooth scrolling
- Mobile-friendly design

**Technologies:** HTML5, CSS3, JavaScript

---

### ⏱️ Task 2 – Stopwatch Web Application
A functional stopwatch application with:
- Start, Pause, and Reset functionality
- Lap time recording
- Clean and responsive UI
- Accurate time tracking

**Technologies:** HTML5, CSS3, JavaScript

---

### 🎮 Task 3 – Tic Tac Toe Web Application
An interactive Tic Tac Toe game including:
- Two-player gameplay
- Win and draw detection
- Restart game option
- Responsive interface

**Technologies:** HTML5, CSS3, JavaScript

---

### 📝 Task 4 – To-Do Web Application
A productivity application featuring:
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Task organization with a clean UI

**Technologies:** HTML5, CSS3, JavaScript
